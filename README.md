## Task 1: Count Arguments
**Description:**  
Implement `countArgs(args...)` that returns how many arguments were passed.

## Task 2: Print All Arguments (One per Line)
**Description:**  
Implement `printAll(args...)` that prints each argument on a new line.  
Give two different implementations.

---

## Task 3: Sum All Arguments
**Description:**  
Implement `sum(args...)` that returns the sum of all arguments.  
Give two different implementations.

---

## Task 4: Check All Arguments Are True
**Description:**  
Implement `allTrue(args...)` that returns `true` only if all arguments evaluate to `true`.  
Give two different implementations.

---

## Task 5: Check Any Argument Is True
**Description:**  
Implement `anyTrue(args...)` that returns `true` if at least one argument evaluates to `true`.  
Give two different implementations.

---

## Task 6: Find Minimum Value
**Description:**  
Implement `minValue(args...)` that returns the smallest value among the arguments.  
Give two different implementations.
